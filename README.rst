=====
Scarf
=====

|IMG1|

.. |IMG1| image:: logo.png
    :width: 50%

Scarf enables analysis single-cell data for millions of cells on a regular laptop.
Currently supports scRNA-Seq, scATAC-Seq and CITE-Seq
